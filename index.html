Branch Command

git branch - to check the branch

git branch -M main - to rename the branch

git checkout BranchName - to switch the branch (to navigate)

git checkout -b NewBranchName - to create a new branch

git merge BranchName - to merge the branch

git branch -d BranchName - to delete the branch

git remote -v

git remote rm origin

git remote add origin https://github.com/meghanapatel7393-hub/apnacollege-demo.git

git remote -v

git push -u origin main - to push the branch
